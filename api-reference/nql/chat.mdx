---
title: "Natural Language Query (NLQ)"
openapi: "POST /api/chat"
description: "Query biomarker data using plain language (e.g. “how did my sleep improve this week?”)."
---

<RequestExample>

```bash HTTP
POST /api/chat HTTP/1.1
Host: api.healthware.dev
Content-Type: application/json
x-api-key: YOUR_API_KEY
dev-id:  YOUR_DEV_ID

{
  "user_connection_id": "USER_CONN_ID",
  "prompt": "what is my recovery score for today",
  "model_provider": "openai",
  "model_name": "gpt-4o",
  "schema": {
    "type": "object",
    "properties": {
      "recovery_score": { "type": "string" }
    }
  }
}
```


```bash BASH
curl -X POST "https://api.healthware.dev/api/chat" \
  -H "Content-Type: application/json" \
  -H "x-api-key: $API_KEY" \
  -H "dev-id:  $DEV_ID" \
  -d @nlq.json   # save the JSON above as nlq.json
```


```javascript JavaScript
const body = {
  user_connection_id: "USER_CONN_ID",
  prompt: "what is my recovery score for today",
  model_provider: "openai",
  model_name: "gpt-4o",
  schema: { type: "object", properties: { recovery_score: { type: "string" } } }
};

const data = await fetch(`https://api.healthware.dev/api/chat`, {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": API_KEY,
    "dev-id":   DEV_ID
  },
  body: JSON.stringify(body)
}).then(r => r.json());
```

</RequestExample>
