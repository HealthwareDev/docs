<RequestExample>

```bash HTTP
GET /api/biomarker/readiness?user_connection_id=USER_CONN_ID&start=2025-07-09T00:00:00Z&end=2025-07-12T00:00:00Z HTTP/1.1
Host: api.healthware.dev
x-api-key: YOUR_API_KEY
dev-id:  YOUR_DEV_ID
```


```bash BASH
curl "https://api.healthware.dev/api/biomarker/readiness?user_connection_id=$CONN_ID&start=2025-07-09T00:00:00Z&end=2025-07-12T00:00:00Z" \
  -H "x-api-key: $API_KEY" \
  -H "dev-id:  $DEV_ID"
```


```javascript JavaScript
const qs = new URLSearchParams({
  user_connection_id: CONN_ID,
  start: "2025-07-09T00:00:00Z",
  end:   "2025-07-12T00:00:00Z"
});

const biomarker = await fetch(`https://api.healthware.dev/api/biomarker/readiness?${qs}`, {
  headers: { "x-api-key": API_KEY, "dev-id": DEV_ID }
}).then(r => r.json());
```

</RequestExample>
