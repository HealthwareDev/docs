<RequestExample>

```bash HTTP
POST /api/oauth/authorize/whoop HTTP/1.1
Host: api.healthware.dev
Content-Type: application/json
x-api-key: YOUR_API_KEY
dev-id:  YOUR_DEV_ID

{
  "developer_user_id": "YOUR_USER_ID"
}
```


```bash BASH
curl -X POST "https://api.healthware.dev/api/oauth/authorize/whoop" \
  -H "Content-Type: application/json" \
  -H "x-api-key: $API_KEY" \
  -H "dev-id:  $DEV_ID" \
  -d '{"developer_user_id":"YOUR_USER_ID"}'
```


```javascript JavaScript
const res = await fetch(`https://api.healthware.dev/api/oauth/authorize/whoop`, {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": process.env.API_KEY,
    "dev-id":   process.env.DEV_ID
  },
  body: JSON.stringify({ developer_user_id: "YOUR_USER_ID" })
});
```
